URM evaluator

(about)

Available URM instructions

Zero
Z(n) sets rn = 0
Successor
S(n) increments rn, giving rn = rn + 1
Transfer (copy)
T(m, n) (or C(m, n)) sets rn = rm
Jump
J(m, n, p) jumps to instruction p if rm = rn
